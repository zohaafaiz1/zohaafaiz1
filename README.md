## Projects

**[StudyPlan AI](https://github.com/zohaafaiz1/studyplan-ai)** — ML academic planner  
Gradient Boosting pipeline predicting student scores (0.71 MAE, 0.74 R²). Applied SHAP and DiCE counterfactual analysis to surface explainable, bias-aware study recommendations across 34 engineered features. Deployed via Streamlit.

**[EV Data Warehouse](https://github.com/zohaafaiz1/ev-data-warehouse)** — large-scale analytics  
Star-schema PostgreSQL warehouse ingesting 100K+ EV adoption records via automated ETL. Built Power BI dashboards for trend analysis. Applied incremental loading and query optimization for efficient reporting at scale.

**[AI Itinerary Generator](https://github.com/zohaafaiz1/ai-itinerary-generator)** — NLP research  
Benchmarked Google Flan-T5 on customized travel plan generation. Curated 1,200 preference-itinerary pairs. Zero-shot prompting outperformed fine-tuning by +10.39 ROUGE-L. Full ROUGE/BLEU error taxonomy documented.

**[AI Presenter Pro](https://github.com/zohaafaiz1/ai-presenter-pro)** — computer vision  
Real-time presenter posture analysis using OpenCV + Mediapipe. Multi-threaded pipeline achieving 30+ FPS with minimal CPU overhead. Modular OOP architecture for API integration.

**[Study Sync](https://github.com/zohaafaiz1/study-sync)** — full-stack desktop app  
All-in-one study tool (to-do, timetable, Pomodoro, notes) built in C# WinForms with ChatGPT API integration. Production-quality Windows application with persistent data management.
